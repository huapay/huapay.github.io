# Huapay.jp

https://www.huapay.jp のリポジトリです。Jekyllで実装されています。

# Requirements

- Ruby
    - 2.5.3
- Bundler gem
    - 1.17.1

## Getting Started

1. `git clone https://github.com/huapay/huapay.github.io.git`
1. `cd huapay.github.io`
1. `bundle --path vendor/bundle`
1. `bundle exec jekyll serve  --livereload`
